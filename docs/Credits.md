## Key Dependencies

| Package                  | Purpose                                     |
|--------------------------|---------------------------------------------|
| React 18                 | Frontend UI framework                       |
| Vite 6                   | Frontend dev server and bundler             |
| React Router v6          | Client-side routing                         |
| Express 4                | Backend API server                          |
| @libsql/client           | SQLite database driver                      |
| Drizzle ORM v0.45.2      | Database schema and query builder           |
| bcryptjs                 | Password hashing                            |
| jsonwebtoken             | JWT generation and verification             |
| cookie-parser            | Reading HttpOnly auth cookies               |
| cors                     | Cross-origin resource sharing               |
| dotenv                   | Environment variable management             |

## Inspiration

The budget template structure is inspired by Showbiz Budgeting, the industry-standard film budgeting software. OSFPM's version is open-source and web-based.
